Eclipse IDE'de Kullanılmayan Importların Otomatik Kaldırılması

Oct 11, 2016

Proje geliştirirken, kod yazarken yazdığımız sınıfları daha sonra güncellememiz gerekebiliyor. Kodları baka sınıflara bölebiliyoruz, kodun daha kısa halini bulabilioyurz. Bu gibi durumlardan ötürü birkaç tane kullanılmayan importlar birikebiliyor. Bu yazıda ben de Java dosyasını kaydettiğimiz anda kullanılmayan bu importların otomatik olarak kaldırılması için Eclipse IDE’de ne yapmamız gerekiyor onu anlatmaya çalışacağım.

Şimdi Eclipse IDE’mizde Mac için üst menüdeki Eclipse kısmına, Windows/Linux için Windows -> Preferences kısmına tıklayalım.

Gelen Preferences ekranında arama kutusuna import yazalım. Gelen seçeneklerden Save Actions‘a tıklayalım.

Akabinde sağ taraftaki ekrandaki Perform the selected actions on save kısmındaki tiki işaretleyim. Bu işaretlemenin ardından Additional Actions kısmındaki tiki de işaretleyelim.

Additional actions tikini işaretledikten sonra sağ taraftaki Configure butonuna tıklayalım. Açılan ekranda Unnecessary Code tabına tıklayalım ve Remove Unused Imports seçeneğine tik koyalım.

OK butonuna tıklayalım ve geri döndüğümüz ekrandaki Apply ve OK butonlarına tıklayalım. Artık gereksiz importlar Java dosyasını kaydettiğimizde otomatik olarak silinecek. Hemen bunu pratiğe dökelim.
