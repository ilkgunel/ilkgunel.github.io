Use Beatiful 3D Leopard Dock in OS X El Capitan

Sep 10, 2016

Hello everyone. In this tutorial i am gonna show you how to you change El Capitan Dock with so beatiful 3D Leopard Dock. Let’s start.

First of all you need to download cDock app to do this purpose. You can download it from here:http://w0lfschild.github.io/app_cDock.html

cDock an OS X application that you can change your dock to you want with it. When the app installation, cDock wants a permission about SIMBL. You need to give this permission to run the program. And do not forget to put cDock app to Applications folder.

After cDock installation , you need to download Leopard Dock files from Deviantart. You can download rar file from here: http://gotier.deviantart.com/art/Leopard-Dock-3D-Original-1-0-83204400

Now, go to your Applications folder, find cDock and right click and click “Show Package Contents”.

Now you see the “Contents” folder. Open it and open Resources folder. In Resources folder open themes folder and create a new folder. You can set a name that you want. I am using M5 string as folder name.

Now extract the rar file open the folder. In this folder you will see the dock and indicator png files.

Now change the name of indicator1.png file to indicator_large.png and move to M5 folder. After this, you should get 4 copy from file that’s name is indicator.png. Names of 4 copy files are

  • indicator_medium_simple.png
  • indicator_medium.png
  • indicator_small_simple.png
  • indicator_small.png

Move this 4 files to M5 folder. After, change bg.png file name to background.png and move to M5 folder.

Lastly you need a .plist file. Create a plist file that’s name is M5.plist and copy these lines to the file.

In the folder, background1.png, indicator_medium1.png and indicator_large.png files my testing file. You can ignore these files.

Now almost finished. Open cDock App and choose the “M5” from Theme as dock option and check “Use image for application Indicators” . If dock doesn’t change, click the Restart Dock button. Again if dock doesn’t change, from Activity Monitor stop cDock and OS X Dock app. Try again. Sometimes cDock freezes.
